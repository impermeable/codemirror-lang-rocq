# CodeMirror Rocq language support

This package implements support for Rocq in CodeMirror 6 editors

### Credits

The Rocq mode powering this package was originally developed by Valentin Robert, Benoît Pin, Emilio J. Gallego Arias, H. Guo, Shachar Itzhaky, and others for use in jsCoq (available here: https://coq.vercel.app/). The original source code can be found in their repository available at https://github.com/jscoq/jscoq (specifically https://github.com/jscoq/jscoq/blob/c6773c29ce23d830d1a903560c6282acb92b203c/frontend/classic/js/mode/rocq-cm6.ts).
